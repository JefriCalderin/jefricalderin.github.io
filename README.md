# JemcoTech Soluciones

Taller técnico especializado en reparación de consolas, controles y electrónica en Barranquilla, Colombia.

## Servicios

- **PlayStation 4** - Reparación integral (lectores ópticos, HDMI, overheating, discos duros, fuentes)
- **PlayStation 5** - Diagnóstico y reparación (arranque, HDMI, refrigeración, SSD)
- **Controles PS3/PS4/PS5** - Drift, botones, vibración, carga, bluetooth
- **Controles Xbox Series** - Drift de joystick, botones, carga USB, rumble
- **Laptops Gamer** - Cambio de pasta térmica, pantallas, teclados, tarjetas de video, upgrades RAM/SSD
- **Tarjetas Electrónicas AX150** - Reparación de equipos con electrónica industrial/doméstica
